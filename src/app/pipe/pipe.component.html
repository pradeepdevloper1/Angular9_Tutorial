Angular Pipe

Pipe are used to transform data

pipe will take data and transform into desired format

Pipe are written using opereator(|)

Types of Pipe

1.Built in Pipe

Lowercase

Uppercase

Currency

Date

2.Parameterized Pipe

we can pass one or more parameter to pipe

3. Chaining Pipe

we can connect multiple pipe to data input

4. Custom Pipe

we can create our own custom pipe for various data formating

BuiltIn Pipe

Name:{{user.firstName|uppercase}},{{user.lastName|lowercase}}
DOB:{{user.DOB|date:'yyyy MM dd'}}
DOB:{{user.DOB|date:'fullDate'}}
DOB:{{user.DOB|date:'shortTime'}}
DOB:{{user.DOB|date:'d/M/y'}}
Salary:{{user.Salary|number:'3.1-2'}}
Salary:{{user.Salary|currency}}
Salary:{{user.Salary|currency:'INR'}}
Salary:{{user.Salary|currency:'INR':'code'}}
json,{{jsonVal|json}}
json,{{user|json}}
percent:{{0.125456|percent:'2.1-2'}}
{{[1,2,3,4,5,6]|slice:1:3}}

Parameterized Pipe

DOB:{{user.DOB|date:'short'}}
DOB:{{user.DOB|date:'long'}}

Chaining Pipe

DOB:{{user.DOB|date:'MMMM'|uppercase}}
